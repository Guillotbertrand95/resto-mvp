# Resto Stock App

Application SaaS de gestion de stock et de pilotage pour restaurateurs.

## 🎯 Objectif

Créer un outil simple et puissant pour :

- gérer les stocks
- suivre les produits
- anticiper les besoins de réapprovisionnement
- analyser les coûts et les marges

## 🧱 Stack technique

- Frontend : React (Vite)
- Backend : Node.js / Express
- Base de données : Supabase (PostgreSQL)
- ORM : Prisma
- Automatisation (prévu) : n8n
- IA (prévu) : analyse & reporting

## 🚀 Lancer le projet

### Backend

```bash
cd backend
npm install
npm run dev
```
